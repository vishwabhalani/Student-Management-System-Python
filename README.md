# Student-Management-System-Python
A console-based Student Management System developed in Python to perform basic student record operations efficiently. This project demonstrates the use of core Python concepts such as functions, lists, dictionaries, loops, and conditional statements.

## Features
- Add new student records
- View all student details
- Search students using Roll Number
- Delete student records
- Interactive menu-driven interface

## Technologies Used
- Python
- Lists and Dictionaries
- Functions
- Conditional Statements
- Loops

## Project Structure
The program stores student information in a list of dictionaries and allows users to manage records through a simple command-line interface.
Each student record contains:
- Student Name
- Roll Number
- Course

## Learning Objectives
This project was created to practice:
- Python programming fundamentals
- Data handling using lists and dictionaries
- Function-based programming
- Building menu-driven applications
